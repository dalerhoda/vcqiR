# VCQI Documentation 

The documents in this folder introduce users to the Vaccination Coverage Quality Indicators (VCQI) programs. 

- Getting Started with VCQI for R Users: provides guidance on installing the vcqiR package and running VCQI for the first time 
- VCQI R User's Guide: provides detailed definitions of VCQI's indicators and describes the flexible features of VCQI that users may adjust by editing blocks B, D, and F of a control program 
- VCQI Forms and Variable Lists: describes datasets that are compatible with VCQI 
- VCQI Interpretation Quick Reference Guide: defines the VCQI indicators, interprets analysis results in English language sentences, and provides interpretation guidance for the plots that VCQI produces 
- VCQI Indicator List with Specifications: describes a set of vaccination program performance measures that can be calculated from household survey data
